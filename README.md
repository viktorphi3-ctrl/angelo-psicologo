# Angelo - Psicólogo e Psicanalista

Website profissional desenvolvido com **Astro**, **React**, **Tailwind CSS** e **Framer Motion**.

## Stack Tecnológica

-   **Framework:** Astro 5.0
-   **Estilização:** Tailwind CSS v4
-   **Iconografia:** Lucide React
-   **Animações:** Framer Motion
-   **Fontes:** Playfair Display (Títulos) & Inter (Corpo)

## Funcionalidades

-   Design minimalista e responsivo (Mobile First)
-   Animações suaves de entrada (FadeIn)
-   Dark/Light mode ready (baseado em variáveis CSS)
-   SEO otimizado
-   Integração com WhatsApp

## Como rodar localmente

1.  Clone o repositório:
    ```bash
    git clone https://github.com/viktorphi3-ctrl/angelo-psicologo.git
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Rode o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

## Estrutura do Projeto

-   `src/components`: Componentes reutilizáveis (Hero, Header, Footer, etc.)
-   `src/layouts`: Layout base da aplicação
-   `src/pages`: Páginas da aplicação (index.astro)
-   `src/styles`: Arquivos CSS globais e configuração do Tailwind v4
-   `src/components/react`: Componentes React (ex: FadeIn)
